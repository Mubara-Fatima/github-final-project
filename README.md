# Simple Interest Calculator

A simple Bash-based calculator that computes simple interest based on user input.

## Features
- Takes principal amount (P), rate of interest (R), and time period (T) as input
- Calculates simple interest using the formula: SI = (P × R × T) / 100
- Displays the calculated result to the user

## How to Use
1. Run the `simple-interest.sh` script
2. Enter the principal amount when prompted
3. Enter the rate of interest
4. Enter the time period in years
5. View the calculated simple interest displayed on screen
